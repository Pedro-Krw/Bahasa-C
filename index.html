Apa Itu Tipe Data !

Yang Pertama Kita tahu bahwa sebuah program di susun untuk memproses atau mengolah data
dan data inilah yang merupakan bagian terpenting dalam sebuah program.
dan Data tebagi menjadi beberapa jenis seperti angka, text, karakter dll...

Tipe data dalam Bahasa C sendiri di bagi menjadi dua kelompok yaitu Primitive dan Composite

Tipe Data Primitive merupakan tipe data yang sudah bisa langsung di gunakan dalam Bahasa C ada tiga tipe data di dalamnya yaitu Numeric, Charater, Boolean

Tipe Data Composite merupakan gabungan dari beberapa tipe data Primitive yaitu contohnya Array, Record dll...
